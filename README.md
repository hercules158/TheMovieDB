# TheMovieDB
App built using the API The Movie DB to find and show movies and Tv Shows, the App deliver to user a overview, release date, rating and a Trailer from Youtube inside the app and he is able to share the trailer in the App he wants. The user is able to search the movie or TV Show he wants, he can add the title he wants in his watch later list. The app is very minimalist and easy to use.
